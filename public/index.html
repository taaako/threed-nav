自分で作成しても良し

あなたが用意した3Dキャラクターに命を吹きこむReact Component

You may create it yourself. or use free materials React component to bring your 3D characters to life.

That is "3D nav UI".

GET STARTED

Installation

$ npm install threed-nav

Usage

import React from 'react';
import ReactDOM from 'react-dom';
import Model from './Model';

ReactDOM.render(
    <React.StrictMode>
        <Model />
    </React.StrictMode>,
    document.getElementById('root')
);
